# 快速开始指南

## 5分钟快速上手

### 前置条件
- Node.js v14+
- 现代浏览器
- M20机器狗（已开启，IP: 10.21.31.103）

### 步骤1: 启动后端（1分钟）

```bash
cd backend
npm install
npm start
```

看到以下输出表示成功：
```
[Server] WebSocket服务器已启动，监听端口 8080
[Server] 控制服务器已启动
```

### 步骤2: 打开前端（1分钟）

在浏览器中打开 `index.html` 文件

或使用本地服务器：
```bash
# 新开一个终端
python -m http.server 8000
# 访问 http://localhost:8000
```

### 步骤3: 连接M20（1分钟）

1. 确认M20地址正确（默认 10.21.31.103:30001）
2. 点击"连接"按钮
3. 等待顶部显示 ✓ 已连接

### 步骤4: 开始控制（2分钟）

**运动控制**:
- 点击"行走"按钮
- 拖拽虚拟摇杆控制方向

**导航任务**:
- 输入目标坐标（如 X=5.0, Y=3.0）
- 点击"开始导航"

**灯光控制**:
- 点击"前灯 OFF"开启前灯

## 常见问题

### Q: 后端启动失败？
A: 检查Node.js版本 `node --version`，需要v14+

### Q: 前端显示未连接？
A: 确保后端已启动，检查浏览器控制台错误

### Q: 无法连接M20？
A: 检查M20是否开启，IP地址是否正确

### Q: 虚拟摇杆不工作？
A: 确保机器狗处于"行走"状态
