## 🔑 核心功能

### 1. 状态查询
- 运动状态、电池电量、当前步态
- 位置信息（X、Y坐标）
- 导航状态、温度监测
- 实时更新显示

### 2. 运动控制
- 运动模式切换（标准/敏捷）
- 运动状态控制（站立/行走/趴下）
- 步态选择（5种步态）
- 虚拟摇杆（20Hz发送频率）
- 灯光控制（前后灯）

### 3. 导航控制
- 导航任务下发（完整参数配置）
- 定位初始化
- 自主充电控制
- 任务取消

### 4. 系统特性
- 自动重连（2秒间隔，最多10次）
- 心跳保活（5秒间隔）
- 实时通信日志
- 响应式设计

## 🚀 使用流程

### 启动
```bash
# 1. 启动后端
cd backend
npm install
npm start

# 2. 打开前端
在浏览器打开 index.html

# 3. 连接M20
点击"连接"按钮
```

### 操作
1. 修改M20地址（如需要）
2. 点击"连接"按钮
3. 等待连接成功
4. 使用各个控制面板

---

## 🔄 通信流程

### 连接建立
```
前端 → 后端: WebSocket连接
后端 → M20: TCP连接
M20 → 后端: 连接确认
后端 → 前端: 连接成功通知
```

### 命令执行
```
前端 → 后端: 命令(JSON)
后端 → M20: 命令(二进制APDU)
M20 → 后端: 响应(二进制APDU)
后端 → 前端: 响应(JSON)
```

### 心跳保活
```
前端 → 后端: 心跳(5秒)
后端 → M20: 心跳(5秒)
```
